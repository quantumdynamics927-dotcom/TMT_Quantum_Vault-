# Code of Conduct

TMT Quantum Vault is committed to a respectful, thoughtful, and safe community
for contributors, users, and researchers.

## Our Expectations

Participants in this project are expected to:

- Be respectful in discussions, code review, and issue triage
- Assume good intent while giving clear, constructive feedback
- Focus on evidence, reproducibility, and the impact of proposed changes
- Respect differing levels of experience, background, and perspective
- Help keep the project aligned with its published ethics and security policies

## Unacceptable Behavior

The following behaviors are not acceptable in project spaces:

- Harassment, personal attacks, or discriminatory language
- Trolling, intimidation, or sustained disruption
- Sharing private information without permission
- Encouraging unsafe, unethical, or harmful use of the project
- Knowingly misrepresenting results, provenance, or validation status

## Scope

This code of conduct applies to:

- GitHub issues, pull requests, and review discussions
- Documentation updates and repository governance discussions
- Any project-managed communication spaces linked from this repository

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers at
`quantumdynamics927@gmail.com`. Include as much context as you can so the report
can be reviewed fairly and promptly.

## Enforcement

Maintainers may take any action they deem appropriate, including warning,
content removal, temporary restrictions, or permanent removal from community
spaces, depending on the severity and pattern of behavior.
